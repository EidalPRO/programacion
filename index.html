FUNDAMENTOS BASICOS DE LA PROGRAMACIÓN

Conceptos fundamentales

¿Qué es la programación?

La programación es el acto de organizar una secuencia de pasos ordenados para llevar a cabo una tarea específica. En el contexto de la informática, la programación se refiere a la creación de programas o aplicaciones mediante el desarrollo de un código fuente. Este código, escrito en lenguajes de programación, permite que los ordenadores ejecuten tareas solicitadas por los usuarios.

¿Como funciona la programación?

La programación funciona mediante la creación de instrucciones precisas que un ordenador puede entender y ejecutar para llevar a cabo una tarea específica. Tales como:

  • Comprensión del problema:

    En primer lugar, el programador debe comprender claramente el problema que está tratando de resolver. Esto implica identificar qué tarea se quiere realizar y cuáles son los requisitos y restricciones asociadas.

  • Diseño del algoritmo:

    Luego, el programador diseña un algoritmo, que es una secuencia de pasos lógicos y ordenados que resuelven el problema identificado. El algoritmo describe la lógica de cómo se debe realizar la tarea.

  • Escritura del código:

    Con el algoritmo en mente, el programador escribe el código fuente utilizando un lenguaje de programación específico. El código fuente es un conjunto de instrucciones escritas en un formato comprensible por humanos.

  • Compilación o interpretación:

    Dependiendo del lenguaje de programación utilizado, el código fuente se compila o se interpreta. En la compilación, el código se traduce en un lenguaje de bajo nivel (código máquina) que el ordenador puede entender directamente. En la interpretación, el código se lee y ejecuta línea por línea por un programa llamado intérprete

  • Ejecución del programa:

    Una vez compilado o interpretado, el programa resultante se ejecuta en el ordenador. El ordenador sigue las instrucciones proporcionadas por el programa y realiza las operaciones necesarias para completar la tarea especificada.

  • Depuración y pruebas:

    Después de la ejecución inicial, el programador puede depurar el programa para corregir errores y realizar pruebas para asegurarse de que funcione correctamente en una variedad de situaciones y con diferentes conjuntos de datos.

Lenguajes de programación

Un lenguaje de programación es un conjunto de reglas y símbolos que permiten a los programadores escribir instrucciones que un ordenador puede entender y ejecutar. Estos lenguajes están diseñados para comunicar órdenes específicas a una computadora, permitiendo así la creación de programas y aplicaciones de software.

Los lenguajes de programación están compuestos por palabras clave, símbolos, reglas de sintaxis y estructuras de datos que definen cómo se deben escribir y organizar las instrucciones para realizar una tarea determinada. Estas instrucciones pueden variar desde operaciones simples como sumar números hasta algoritmos complejos para resolver problemas complejos.

Los lenguajes de programación pueden clasificarse en diferentes categorías según su nivel de abstracción y su propósito, como:

  • Lenguajes de bajo nivel:

    Estos lenguajes están más cerca del lenguaje máquina y están diseñados para interactuar directamente con el hardware de la computadora. Ejemplos incluyen el lenguaje ensamblador.

  • Lenguajes de alto nivel:

    Estos lenguajes están más cerca del lenguaje humano y se centran en la eficiencia del desarrollo y la legibilidad del código. Ejemplos populares incluyen Python, Java, C++, JavaScript, y muchos otros.

  • Lenguajes de programación orientados a dominios específicos (DSL):

    Estos lenguajes están diseñados para resolver problemas específicos en un dominio particular, como SQL para bases de datos o HTML para páginas web.

Diagramas de flujo


Un diagrama de flujo es una representación gráfica de un proceso o algoritmo que utiliza símbolos y flechas para mostrar la secuencia de pasos necesarios para completar una tarea. Es una herramienta visual que ayuda a comprender y comunicar de manera clara y concisa la lógica de un proceso o procedimiento.

Los diagramas de flujo suelen utilizarse en el campo de la programación, la ingeniería, la gestión de proyectos y otras áreas donde es importante entender y documentar la secuencia de pasos de un proceso. Algunos de los símbolos más comunes utilizados en los diagramas de flujo incluyen:



Representa el punto de inicio o finalización del proceso. Generalmente se representa como un óvalo .

Representa una acción o tarea que se realiza en el proceso. Se representa como un rectángulo.

Representa una bifurcación en el flujo del proceso, donde se toma una decisión basada en una condición. Se representa como un rombo.

Se utiliza para conectar partes de un diagrama de flujo que están ubicadas en diferentes páginas.

Representa la entrada de datos al proceso o la salida de datos del proceso. Se representa como un paralelogramo.
